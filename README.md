# node-survivio

(Note: This module is in development)

An Node CLI for surviv.io. You can:

- Read global stats
- Read user stats

## Global stat types

- Most kills
  - today/week/all
  - solo/duo/squad

## User stats

- Games today
- Kills today
- Kills last match
- Total games
- Total kills
